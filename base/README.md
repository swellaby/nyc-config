# nyc-config 
Our base configuration for [nyc][nyc-pkg-url]. Geared for our specific needs, but feel free to use it if it helps for you!

## Configuration
- `reporters`  
Adds html, lcov, cobertura, text, and text-summary reporters
- `cache`: true
- `all`: true
- `per-file`: true
